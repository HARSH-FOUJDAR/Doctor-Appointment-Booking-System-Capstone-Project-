# Doctor Appointment Booking System

A full-stack Doctor Appointment Booking System that allows patients to book appointments, doctors to manage schedules, and administrators to manage the platform efficiently.

## Live Demo

🔗 https://bejewelled-arithmetic-643bb6.netlify.app.


## Features

* User Authentication & Authorization (JWT)
* Role-Based Access Control (Admin, Doctor, Patient)
* Doctor Management
* Appointment Booking & Cancellation
* Patient Dashboard
* Doctor Dashboard
* Admin Dashboard
* Online Payment Integration
* Appointment Status Tracking
* Responsive Design

## Tech Stack

### Frontend

* React.js
* Redux Toolkit
* Tailwind CSS

### Backend

* Node.js
* Express.js

### Database

* MongoDB

### Authentication

* JWT (JSON Web Tokens)

### Payment Gateway

* Razorpay / Stripe


## Project Modules

### Patient Panel

* Register/Login
* Book Appointments
* Make Online Payments
* View Appointment History

### Doctor Panel

* Manage Availability
* View Patient Appointments
* Update Appointment Status

### Admin Panel

* Manage Doctors
* Manage Patients
* Monitor Appointments
* Platform Administration
